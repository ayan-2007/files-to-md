# files-to-md

Convert your files into clean Markdown format.

`files-to-md` is a simple utility that helps transform different file types into Markdown, making content easier to read, share, version-control, and use with tools like documentation systems, note apps, and AI workflows.

## ✨ Features

- 📄 Convert files into Markdown
- 📝 Generate clean `.md` output
- ⚡ Fast and lightweight
- 🤖 AI-friendly formatting
- 📚 Useful for documentation and knowledge bases
- 🔧 Easy to integrate into workflows
